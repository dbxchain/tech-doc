##### 参数：from\_account to\_account amount DBX "memo" BROADCAST {#参数：fromaccount-toaccount-amount-dbx-memo-broadcast}

| 参数 | 说明 |
| :--- | :--- |
| from\_account | 转帐发起帐户 |
| to\_account | 转帐接收帐户 |
| amount | 转帐数目 |
| DBX | 转帐资产 |
| memo | 转帐备注，交易所用户充值时需要填写备注 |
| true | true表示真正执行 |
